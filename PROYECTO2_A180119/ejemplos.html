Ejemplos de Frameworks

Existen una gran cantidad de frameworks para
programación.
Por ejemplo:
  1. Flutter

  2. Ionic

  3. React Native

  4. Xamarin

  5. PhoneGap

  6. Native Script

  7. Appcelerator Titanium

  8. jQuery Móvil

  9. Móvil Angular UI

  10. Weex

  11. Corona SDK

  12. Unity

  13. Qt

  14. Intel System Studio

La principal función de estos y otros framework es la de
disponer de un entorno que facilite la escritura de código
y el desarrollo de una aplicación. El framework ofrece
herramientas para organizar y controlar el código
generado, además de permitir ser más eficientes que
utilizando los métodos tradicionales. El uso de un
framework reduce la cantidad de errores cometidos
durante los procesos de programación, facilitando las
labores de los desarrolladores.
